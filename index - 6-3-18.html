Motor

Un motor es la parte sistemática de una máquina capaz de hacer funcionar el sistema.

Transmision

Es el mecanismo encargado de transmitir potencia entre dos o más elementos dentro de una máquina

Filtros

Un filtro de aire es un dispositivo que elimina partículas sólidas como por ejemplo polvo, polen y bacterias del aire.

Frenos

Un freno es un dispositivo utilizado para detener o disminuir la velocidad de algún trasporte.

Suspension

se refiere al conjunto de componentes que conectan las ruedas con el chasis de un vehículo permitiendo el movimiento relativo entre ambos.

Direccion

s el conjunto de órganos que permiten a cualquier vehículo modificar su trayectoria para seguir el rumbo deseado.
